# NextPress Project

NextPress is a basic news portal and has a very simple structure. 

## Features
- Show all posts
- Pagination Posts
- Add recently viewed post to Recently Posts


## Technologies Used
- Nextjs
- TypeScript
- Redux
- Tailwind CSS
